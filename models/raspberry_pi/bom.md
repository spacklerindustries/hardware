# Raspberry Pi Caddy

* 1 x front_panel_raspberry_pi.stl
* 1 x rear_panel.stl
* 2 x handle.stl
* 8 x 2.5mm_spacer.stl (optional/if laser/CNC top/bottom/sides)

* 1 x top_panel.stl
* 2 x side_panel.stl
